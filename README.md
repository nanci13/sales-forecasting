# 📊 Superstore Sales Analysis and Time Series Forecasting

## 📌 Project Overview
This project presents an interactive **Superstore Sales Analysis and Time Series Forecasting Dashboard** developed using **Microsoft Power BI**. The dashboard transforms raw sales data into meaningful business insights through interactive visualizations and forecasting techniques. It enables users to analyze sales performance, profitability, customer behavior, and regional trends while predicting future sales for informed decision-making.

---

## 🎯 Objectives
- Analyze historical sales data to identify business trends.
- Monitor key performance indicators (KPIs).
- Visualize sales, profit, orders, and customer segments.
- Compare performance across regions, categories, and shipping modes.
- Forecast future sales using Power BI's Time Series Forecasting.

---

## 🛠️ Tools & Technologies
